# Phase I: Console Todo Application

## Overview

In-memory Python console application for task management with 5 Basic Level CRUD operations (Create, View, Update, Delete, Mark Complete). Built using spec-driven development with Claude Code and Spec-Kit Plus.

## Requirements

- Python 3.13+
- UV package manager

## Installation

1. Clone the repository:
```bash
git clone <repository-url>
cd "Naz Todo"
```

2. Create and activate virtual environment with UV:
```bash
# Create virtual environment
uv venv

# Activate virtual environment
# On Windows:
.venv\Scripts\activate

# On macOS/Linux:
source .venv/bin/activate
```

3. Install development dependencies:
```bash
uv pip install -e ".[dev]"
```

## Usage

Run the console application:
```bash
python -m src.cli.menu
```

### Features

The application provides a numbered menu with the following operations:

1. **Add Task** - Create new tasks with title and optional description
2. **View Tasks** - Display all tasks with ID, title, description, and completion status
3. **Update Task** - Modify task title or description
4. **Delete Task** - Remove tasks from the list (with confirmation)
5. **Mark Complete/Incomplete** - Toggle task completion status
6. **Exit** - Quit the application (with confirmation)
